### 3 ETL/ELT пайплайна для сбора данных

#### Проект включает в себя 3 пайплайна по загрузке, обработке и сохранению данных в базу данных PostgreSQL

### Описание проекта

etl_csv_country_happiness - загрузка информации из csv файла

walmart_analysis - загрузка информации из csv файла и получение аналитической 
информации из csv с использованием PySpark

scrap_games - получение информации с сайта для скрапинга https://sandbox.oxylabs.io с 
использованием BeautifulSoup и Selenium

### Настройка и запуск
1. Для работы требуется установленное подключение с базой данных PostgreSQL
2. Внести имя пользователя, пароль, и название базы данных в файле data.py
3. Скачать драйвер для соединения PySpark и Postgre
4. Внести путь к файлу в path_to_jar в data
5. Находясь в корневом каталоге проекта установить зависимости командой 

       pip install -r requirements.txt

6. Запуск пайплайнов осуществляется из файлов main.py в папках проектов
